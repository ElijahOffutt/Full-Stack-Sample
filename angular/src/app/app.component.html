The Run Down

How it all comes together.

If you have somehow gotten to this page without installing and running the application, I would recommend going to the github project and thoroughly reading through the installing steps to get things properly configured and running.

The app is built with two files being the main server and the model in which items are interfaced with on the database. Essentially before each call to the server is sent the data is stripped to a general search that is then used on the server to find relating items. That data is then returned and using javascript higher level functions, the data is then filtered even more on the fly.
